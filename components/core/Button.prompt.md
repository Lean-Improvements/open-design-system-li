Primary action button — use `default` (Cursor Orange) for the single main CTA on a view; everything else is `secondary`, `outline` or `ghost`.

```jsx
<Button variant="default">Create epic</Button>
<Button variant="secondary">Cancel</Button>
<Button variant="ghost" size="sm">Skip</Button>
<Button variant="destructive">Delete</Button>
```

Variants: `default` (orange), `secondary` (white card), `outline`, `ghost`, `link`, `destructive`, `ink` (dark inverse). Sizes: `default` (40px), `sm` (36px), `lg` (44px), `icon` (40×40). Keep `default` scarce — one orange CTA per view.
