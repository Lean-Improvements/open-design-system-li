# Security Policy

## Scope

This is a purely presentational design-system package (design tokens, Tailwind
preset, theme CSS and reference UI components). It contains no backend code, no
secrets, and no network calls.

## Reporting a vulnerability

If you believe you have found a security issue (for example, an accidentally
committed secret, or a supply-chain concern), please **do not open a public
issue**. Email **hola@leanimprovements.es** with the details and we will respond as
soon as we can.

Please include:

- A description of the issue and its potential impact.
- Steps to reproduce, or the file/commit where the problem is.

Thank you for helping keep the project and its users safe.
